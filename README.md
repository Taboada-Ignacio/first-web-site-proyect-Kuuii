# Water & Ice Sales Website

This project is a simple informational website built with React
for a local small business dedicated to the sale of water and ice.

The company operates in **Comodoro Rivadavia, Chubut, Argentina**,
and the website is intended to serve as a digital presence to
promote the business and facilitate customer contact.
